# 뱀 게임 (Snake Game)

HTML5 Canvas와 JavaScript로 만든 클래식 뱀 게임입니다.

## 게임 특징

- **HTML5 Canvas** 기반의 부드러운 그래픽
- **모듈화된 JavaScript** 구조 (ES6 모듈)
- **반응형 디자인** (모바일 지원)
- **로컬 스토리지** 최고 점수 저장
- **키보드 컨트롤** 지원

## 게임 방법

1. **방향키**로 뱀을 조종합니다
2. **스페이스바**로 게임을 시작하거나 일시정지합니다
3. 빨간 음식을 먹으면 점수가 올라가고 뱀이 길어집니다
4. 벽이나 자신의 몸에 부딪히면 게임이 종료됩니다

## 프로젝트 구조

```
snake-game/
├── index.html          # 메인 HTML 파일
├── style.css           # 스타일시트
├── js/
│   ├── game.js        # 메인 게임 로직
│   ├── snake.js       # 뱀 클래스
│   └── food.js        # 음식 클래스
├── README.md          # 프로젝트 설명서
└── .gitignore         # Git 제외 파일
```

## 실행 방법

1. 프로젝트 폴더를 다운로드합니다
2. `index.html` 파일을 웹 브라우저에서 엽니다
3. 게임을 즐기세요!

### 로컬 서버 실행 (권장)

```bash
# Python 3 사용
python -m http.server 8000

# Node.js 사용 (npx 필요)
npx http-server

# VS Code Live Server 확장 프로그램 사용
```

## 기술 스택

- **HTML5** - 게임 구조
- **CSS3** - 스타일링 및 애니메이션
- **JavaScript (ES6)** - 게임 로직
- **Canvas API** - 그래픽 렌더링

## 주요 기능

### Snake 클래스
- 뱀의 이동 및 방향 제어
- 성장 및 충돌 감지
- 그래픽 렌더링

### Food 클래스  
- 무작위 위치 생성
- 뱀 몸체와 겹치지 않는 위치 보장
- 원형 그래픽 표현

### Game 클래스
- 게임 루프 관리
- 키보드 입력 처리
- 점수 및 게임 상태 관리
- 로컬 스토리지 연동

## 컨트롤

- **↑** : 위로 이동
- **↓** : 아래로 이동  
- **←** : 왼쪽으로 이동
- **→** : 오른쪽으로 이동
- **Space** : 게임 시작/일시정지
